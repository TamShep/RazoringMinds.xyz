RazoringMinds Privacy Policy

This document entails the privacy policy and agreement that you accept when adding RazoringMinds to your server, or as a member of any server in which RazoringMinds is on.


This privacy policy applies to the following web applications:
• RazoringMinds#1107 - Public Bot
• RazoringMinds Beta#3791 - Non-Public Testing Bot

This privacy policy applies to the following websites:
• RazoringMinds Official Website - https://razoringminds.xyz
• RazoringMinds Older Domain - https://razoringminds.cf

Terminology

• Server Administrator - Any user who has the permission, ability or access to add or configure RazoringMinds to a server.
• Server Member - Any user who is a member of a server which RazoringMinds is present on.
• Service User - Anyone who is authorized to the RazoringMinds Admin Web Panel.

Data Collected By Command

The following may be collected via user run command, and may be entered by the user, voluntarily. This data is not collected automatically. When providing data in this way, you forego any rights to the content of the data provided.
• Server configurations (region, name, id)
• Server ownership
• List of server's members
• List of server's roles
• Server & RazoringMinds Premium status

Data Collected When Features Are Enabled

The following will be collected automatically when a server administrator configures the bot to perform checks for news or another automated feature within RazoringMinds. These features are not enabled automatically, and therefore the following will not be collected automatically unless the corresponding feature is enabled by the server administrator.
• Server configurations (channel names, channel ids)
• List of server's roles
• List of server's members

Data Collected Automatically

The following is collected automatically when RazoringMinds joins a server. This data is used to provide statistics or server information.
• Server configurations (region, name, id, creation date, large status)
• Server ownership
• Server member count
• Server role count
• Server channel count
• Server availability
• List of server's roles
• Server & RazoringMinds Premium status

Data Collected via Authorization

The following is collected when a user has access to the RazoringMinds Admin Web Panel via Discord OAuth2 in order to verify identity and accessibility to admin services on RazoringMinds. (This is only applicable to RazoringMinds Team members, and thus is not shared with anyone.)
• Discord Username
• Discord ID
• Discord Avatar
• Discord Account email address

Data Collected via Website Usage

The following is collected/stored upon visiting the official RazoringMinds websites. This data is used to ensure website security and to comply with local, federal, and regional laws.
• Device Internet Protocol Address
• Device Geolocation Data
• Website Usage Patterns
• Discord OAuth2 data (Discord Username, Discord ID, Discord Avatar, Discord Account email address)
• Browser Information
• Cloudfare History

Data Storage & Protection

All data is stored on secured servers. Maximum efforts are taken to keep collected data, as defined above, protected, but absolute security cannot be guaranteed. We are not liable for any damages or stolen information, in which we collect, from our servers.
Data is held for up to two (2) years after removal of bot from Discord server, up to three (3) years after an unexplained four (4) year inactivity on our website, up to twenty-four (24) hours after request of removal of data (see Service Usage Agreement section), and/or up to seventy-two (72) hours after legal request to remove data.
Unless otherwise mentioned, data of users still actively using our bot/using our website/using our admin panel, is retained for as long as it is needed. We do major data removals every two (2) weeks or upon migration to new servers (if an upgrade occurs).
Cookies may be used to ensure fast login and access to our services. Cookies are automatically removed after one (1) month of inactivity or upon clearing cookies in user's web browser.

Data Shared with Third Parties & Third Party Sites

Some data may be shared with third parties, notably Cloudfare and Google Analytics. The following is the data we send and where it goes to.
• Device Internet Protocol Address and Website Usage Patterns is sent to Cloudfare for DDoS protection and website security.
• Website Usage Data and Browser Information is sent to Google Analytics for usage in determining traffic patterns and keeping website loading times faster.
• Other Data may be sent to Discord depending on what you choose to sent to them when inviting our bot or using Discord's OAuth2 service.
Some third parties may be linked to on our website, we provide the links for your convenience, but we do not review, control, or monitor the privacy practices of web sites operated by others. We are not responsible for the performance of websites operated by third parties or for your history with them. Therefore, whenever leaving our websites, we recommend that you take the time to review each website's privacy practices and make your own decisions regarding the adequacy of these practices.
